Follow these steps to download and run the Employee Attendance System:

1. **Download the Application**
   Download the latest version of the application from the Releases page. You can find it [here](https://github.com/ChetanAnekar/employee-attendance-system-portfolio/releases).

   [![Download](https://img.shields.io/badge/Download%20Now-brightgreen)](https://github.com/ChetanAnekar/employee-attendance-system-portfolio/releases)

2. **System Requirements**
   - Operating System: Windows, macOS, or Linux.
   - Node.js version 14 or higher.
   - MongoDB Atlas account for database access.
   - Internet connection for real-time tracking.

3. **Install Dependencies**
   - You need to install Node.js if you haven't already. Visit [Node.js Official Website](https://nodejs.org/en/download/) for installation instructions.
   - Clone the repository to your local machine using the following command:
     ```bash
     git clone https://github.com/ChetanAnekar/employee-attendance-system-portfolio.git
     ```
   - Navigate to the project folder:
     ```bash
     cd employee-attendance-system-portfolio
     ```
   - Install necessary packages by running:
     ```bash
     npm install
     ```

4. **Configuration**
   - Create an `.env` file in the project root. Copy the sample configuration provided and fill in your MongoDB credentials. This step is essential for connecting to your database.
   - To get your Cloudinary API credentials, create an account on [Cloudinary](https://cloudinary.com/) and retrieve your API keys.

5. **Run the Application**
   - Start the server with the following command:
     ```bash
     npm start
     ```
   - Open your web browser and visit `http://localhost:3000` to access the application.

6. **Usage**
   - Log in using your credentials.
   - Use the attendance dashboard to track employee presence.
   - Utilize geofencing features for real-time updates.

7. **Documentation**
